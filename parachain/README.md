# Parachain Runbooks

Chain-level debugging for parachain operations — registration, block inclusion, messaging, coretime.

For collator **node-level** issues (sync, peers, resources), see [node/collator-operations](../node/collator-operations.md).

## Runbooks

| Runbook | When to use |
|---|---|
| [not-producing](not-producing.md) | Parachain blocks not being included on relay chain |
| [xcm-delivery](xcm-delivery.md) | Cross-chain messages stuck or failing |
| [coretime](coretime.md) | Core assignment, expiry, or scheduling issues |
| [onboarding](onboarding.md) | New parachain registration and genesis |
